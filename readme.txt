Track your calorie burn with our free calorie calculator. Quickly estimate your calories burned based on activity and duration.

[https://www.calculator.io/calories-burned-calculator/](https://www.calculator.io/calories-burned-calculator/)

== Usage ==

Add the Calories Burned Calculator shortcode to your page, post or sidebar:

`[ci_calories_burned_calculator]`

Add the following code to your template where you would like the Calories Burned Calculator to appear:

`<?php display_ci_calories_burned_calculator(); ?>`

== Installation ==

1. Upload the Calories Burned Calculator /ci_calories_burned_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Calories Burned Calculator plugin through the "Plugins" menu in WordPress.
